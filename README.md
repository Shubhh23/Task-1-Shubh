# Task-1-Shubh
Data Cleaning & Preparation Project
Project Overview

This project focuses on cleaning and preparing a raw e-commerce dataset for analysis. The objective was to improve data quality by identifying and resolving missing values, duplicate records, formatting inconsistencies, and calculation errors. The cleaned dataset can be used for reliable business analysis and reporting.

Data Cleaning Tasks Performed
1. Missing Value Handling
Identified null and blank values.
Filled missing categorical values using appropriate methods.
Filled missing numerical values using statistical techniques where required.
2. Duplicate Removal
Checked for duplicate OrderIDs and duplicate records.
Removed duplicate entries to maintain data integrity.
3. Data Format Standardization
Standardized date formats.
Corrected text inconsistencies and extra spaces.
Ensured numerical columns contained valid numeric values.
4. Validation Checks
Verified OrderID uniqueness.
Checked CustomerID formatting.
Validated TotalPrice calculations.
Reviewed categorical fields for spelling inconsistencies.
